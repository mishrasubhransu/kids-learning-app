# Kids ABC & Numbers Learning App

An interactive learning app for kids to learn alphabets and numbers with visual counting.

## Features

- **Alphabet Mode**: Display letters A-Z in large, easy-to-read format
- **Numbers Mode**: Count objects (1-10) with visual representations
- **Multiple Objects**: Choose between peanuts, strawberries, or eggs for counting
- **Keyboard Navigation**: Use left/right arrow keys to navigate
- **Responsive Design**: Works on desktop and mobile devices
- **Progress Indicators**: Visual dots showing current position

## Tech Stack

- React 19
- Vite
- Tailwind CSS 4
- Lucide React Icons

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Usage

1. Select **Alphabets** or **Numbers** mode from the top toggle
2. In Numbers mode, choose what object to count (peanuts, strawberries, or eggs)
3. Navigate using:
   - Left/Right arrow keys
   - Click the chevron buttons on the sides
4. Watch the progress dots at the bottom
